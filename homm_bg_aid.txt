GAME ROUND
----------
    GAME ROUND BEGIN
        - Reset tokens
            - Town (Build, Population, Spell Book)
            - Heroes (Movement, Expert)
        - Round type
            - First    => nothing
            - Resource => all Players gain income
            - Astro    => draw & resolve Astro card
        - Scenario timed events
    ANYTIME DURING GAME ROUND
        - IF 1x Morale => may choose one option & lose 1x Morale
            - Draw 1 card
            - Discard & Draw same number of cards
            - Reroll your one any die
    ANYTIME DURING GAME ROUND EXCEPT COMBAT
        - Build: build one building
        - Population: recruit & reinforce any number of Units OR recruit Secondary Hero
        - Spell Book: draw spells as Mage Guild instructs
    PLAYER TURN
        PLAYER TURN BEGIN
            - Discard any number of cards, then Draw cards up to Hero Limit
        DURING PLAYER TURN        
            - Use 1 MP to move & resolve field
            - Use 1 MP to discover Map Tile OR place Map Tile
              (tile must touch your Hero & 2+ tiles, tile must have path to any tile)
            - Play cards from your hand
        PLAYER TURN END
            - IF 2x Negative Morale => discard all cards & discard 1x Negative Morale
    GAME ROUND END
        - Check Scenarion Win/Lose conditions
        - IF Player plays 3 rounds without Town/Settlement => Lose

HEROES
------
                     MAIN                 SECONDARY (for 10 gold in Town/Settlement)
    Movement Point    3                    2
    Use Deck & Exp    Yes                  No
    Use Units         Yes                  Yes
    Collect Cards     Yes                  Yes
    Avoid combat      Surrender            Remove before combat (Counts as defeat)
    IF surrender =>   Move to Town/Settl   Remove
    IF defeated =>    Move to Town/Settl   Remove


COMBAT
------
    COMBAT BEGIN
        - May remove Secondary Hero (Counts as defeat)                 +---------+
        - IF Hero lvl > Neutrals lvl => Neutrals flee                  |Backline |
        - Use Diplomacy: IF Hero lvl = Neutrals lvl => Neutrals flee   |Frontline|---+
        - IF Combat in Town with Citadel => Gate, Walls, Arrow Tower   | W W G W | A |
        - Units placement: MAX 5 Units                                 |Frontline|---+
                           First Attacking player, then Defending      |Backline |
        - Defending Player may use army in Town/Settlement for 8 gold  +---------+
    COMBAT ROUND
        COMBAT ROUND BEGIN
            - Reset tokens: Activate (Faction Cube), Retaliation (Black Cube)
        PLAYER TURN (Attacker first)
            - 1 spell per round during any your Unit activation
            - Spell damage removes Paralysis token (friendly fire on)
            - IF vs Neutrals => +1 round costs 1 MP
        UNIT TURN
            1. Activate (mark Unit with your Faction Cube)
                a) May surrender for 10 gold IF NOT defending Town
                b) Remove Defense Token (can't use Defense Token this round)
                c) IF Paralysis token => remove Paralysis token & end Unit turn
                d) Use <Activate>, <Ongoing>, <Instant>
            2. Move
                - IF Ground/Flying => Move 0-3 then Attack/Defend
                - IF Ranged => Attack then Move 0-1
                               OR Defend only
                - IF Defense Token used => end turn
            3. Use <Activate>, <Ongoing>, <Instant>
            4. Attack Phase
                a) Attacker - Use <Instant>
                b) Attacker - Roll Attack die (D3: -1, 0, +1)
                            - IF Ranged in melee OR backline to backline => 2x Attack die, resolve lower
                c) Defender - IF Defense token => roll Defense die (D3: 0, 0, +1)
                d) Defender - IF Paralysis token => remove Paralysis token & Retaliate IF can
            5. Resolve damage
                - IF Gate/Wall attacked by Ground/Flying => Gate/Wall destroyed
                - DAMAGE =   Attack stat
                           - Defense stat
                           + Attack die
                           - Defense die
                           - 1 IF ranged attack & enemy Unit behind Wall/Gate
                - IF  DAMAGE < 0  =>  DAMAGE = 0
                - HP = HP - DAMAGE
            6. Retaliation
                - Only IF melee
                - Function like Attack Phase
                - Each Unit retaliates once per Combat Round (mark with Black Cube)
    COMBAT END
        - Discard all <Ongoing> in play
        RETREAT happens IF combat was not extended vs Neutrals except Azure
            - Move Hero to previous map field
            - Keep survived Units
        SURRENDER may happen on your Unit Activation only vs Hero IF NOT defending Town
            - Pay 10 gold to Enemy Hero
            - IF Main Hero surrenders => Move Hero to any your Town/Settlement IF any
            - IF Secondary Hero surrenders => Remove Hero
            - Keep survived Units
        DEFEAT happens IF all Units are defeated
            - IF Main Hero defeated => Gain Negative Morale & Move Hero to any your Town/Settlement
            - IF Main Hero defeated vs Hero => Pay 5 gold to Winner
            - IF Main Hero defeated without Town/Settlement => Lose game
            - IF Main Hero defeated when defending last Town/Settlement => Lose game
            - IF Secondary Hero defeated => Remove Hero
            - Take starting Units
            - Winner Hero gains enemy Faction Cube
            - Winner Hero gains Exp:
                - IF Hero level = Enemy level => +1 Exp
                - IF Hero level < Enemy level => +2 Exp
                - IF Azure enemy => gain 7 level
