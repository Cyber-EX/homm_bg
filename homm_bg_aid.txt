SCENARIO SETUP
--------------
    - Choose Difficulty.
    - Set Environment: Map Tiles, Combat Board, Round Tracker.
    - Set Tokens Piles: Resources, Damage, Paralysis/Defense, Morale, Black Cubes.
    - Set Decks: Ability, Artifact (+ 10 Spell Scrolls), Spell, Neutral Units (+ Models),
                 Astrologer, Event, Pandora’s Box.
    - Set Cards Supply: Statistics, Empowered Statistics, War Machines, Walls, Gate, Arrow Tower,
                        Spell Scrolls, Empowered Abilities, Summoned Units, Creature Bank Units.
    - Set Tokens Supply: Attack, Weakness, Corrosion, Time, Quicksand, Summon, Clone, Whirlpool,
                         Empowered Ability, Creature Bank, Stack, Force Field, Land Mine.
    - Set Battlefield Extras: Obstacles, Fire Wall, Initiative, Morale Decks.
    - Set other Scenario specific features.

PLAYER SETUP
------------
    - Choose Faction and Main Hero.
    - Set Starting Components:
        - Main Hero (+ Model), Town (+ Model).
        - Faction Cubes, Tokens (Build, Population, Spell Book, Movement).
        - Units (+ Models), Buildings, Resources, Income, Bonus.
    - Deck: Statistics, Ability, Specialty I, Magic Arrow (+1 IF Magic Hero).
    - Supply: Secondary Hero Model, Units (+ Model), Buildings, Map Tiles, Specialty IV & VI.


GAME ROUND
----------
    GAME ROUND BEGIN
        - Reset tokens
            - Towns (Build, Population, Spell Book)
            - Heroes (Movement, Expert)
        - Round type
            - IF First    => resolve immediate effects from starting Buildings
            - IF Resource => all Players gain income; draw & resolve Event card
            - IF Astro    => draw & resolve Astrologer card
        - Scenario timed events
    ANYTIME DURING GAME ROUND
        - Use Positive Morale => choose one action:
            - Draw 1 card
            - Discard & Draw same number of cards
            - Reroll your any die
    ANYTIME DURING GAME ROUND EXCEPT COMBAT
        - Build: build one building
        - Population: recruit & reinforce any number of Units
                      OR recruit Secondary Hero
        - Spell Book: draw spells as described in Mage Guild
    PLAYER TURN
        PLAYER TURN BEGIN
            - Discard any number of cards, then Draw cards up to Hero Limit
        DURING PLAYER TURN        
            - Use 1 MP to move & resolve field
            - Use 1 MP to resolve field where Hero is now
            - Use 1 MP to discover Map Tile OR place Map Tile
              (tile must touch your Hero & 2+ tiles; tile must have path to any tile)
            - Play cards from your hand
        PLAYER TURN END
            - IF 2x Negative Morale => discard all cards & lose 1x Negative Morale
    GAME ROUND END
        - Check Scenario Win/Lose conditions
        - IF Player plays 3 rounds without Town/Settlement => Lose game


MAP LOCATIONS
-------------
    - Multi-Flaggable
        - Obelisk: Scenario specific effect.
        - Star Axis: Remove Statistic from Hand, replace with same Empowered Statistic.
    - Flaggable
        - Dragon Utopia: Scenario specific effect.
        - Random Town: Just like Gold Mine, but income is 10 Gold.
                       Guards: Random Faction, 1x Pack of Bronze, 2x Pack of Silver, 2x Few of Gold. 
        - Settlement: Just like Mine, but Player chooses Resource.
                      OR Upgrade one bronze/silver Unit for half price round up (free for first Player)
    - Revisitable
        - Black Market: Look top 4 cards from Artifact Discard, buy one for:
                            - Minor Artifact - 5 Gold
                            - Major Artifact - 7 Gold
                            - Relic Artifact - 10 Gold
        - Library of Enlightenment: Pay 3 Gold to Remove 1 Statistic from Hand or Discard, replace it with other Statistic.
                                    Twice per visit.
        - Sanctuary: Hero cannot be attacked here.
        - Tavern: Pay 7 Gold to gain Secondary Hero & Choose enemy player to discard 1 random card from hand.
        - Trading Post: Trade resources
                        OR Remove card (Variant: gain 1 Gold for each Removed card)
                        OR Buy War Machine
        - War Machine Factory: Buy War Machine
    - Visitable
        - Hill Fort: Reinforce one Bronze/Silver Unit with discount 3 Gold.
        - Magic Spring: Look at top 3 Cards of your Discard, return one to Hand.
                        Return remaining to top of your Discard in any order.
        - Market of Time: Remove one card from hand, then Search (2) Ability, Spell or Artifact.
        - Prison: Gain Secondary Hero. If you have Secondary Hero, gain 3 Gold.
        - Redwood Observatory: Discover Map Tile adjacent to this.
        - Scholar: Roll 1 Attack Die.
                   +1 - Gain any Statistic or Remove Statistic from hand.
                    0 - Draw 2 Cards from Ability, gain one, discard other.
                   -1 - Draw 2 Cards from Spell, gain one, discard other.
        - University: Pay 6 Gold to Search (4) Ability Discard.
        - Grail: Gain a Grail. Only one Grail exist. Scenario specific effect.
        - Tree of Knowledge: Pay 3 Crystals or 10 Gold to gain 2 Exp.
        - Witch Hut: Remove Ability from Hand
                     OR Look top card of Ability, put it into Hand or into Ability Discard.

HEROES
------
                     MAIN                 SECONDARY (for 10 gold in Town/Settlement)
    Movement Point    3                    2
    Use Deck & Exp    Yes                  No
    Use Units         Yes                  Yes
    Collect Cards     Yes                  Yes
    Avoid combat      Surrender            Remove before combat (Counts as defeat)
    IF surrender =>   Move to Town/Settl   Remove
    IF defeated =>    Move to Town/Settl   Remove


COMBAT
------
    COMBAT BEGIN
        - May remove Secondary Hero (Counts as defeat)                 +---------+
        - IF Hero lvl > Neutrals lvl => Neutrals flee                  |Backline |
        - Use Diplomacy: IF Hero lvl = Neutrals lvl => Neutrals flee   |Frontline|---+
        - IF Combat in Town with Citadel => Gate, Walls, Arrow Tower   | W W G W | A |
        - Units placement: MAX 5 Units                                 |Frontline|---+
                           First Attacking player, then Defending      |Backline |
        - Defending Player may use army in Town/Settlement for 8 gold  +---------+
    COMBAT ROUND
        COMBAT ROUND BEGIN
            - Reset tokens: Activate (Faction Cube), Retaliation (Black Cube)
        PLAYER TURN (Attacker first)
            - 1 spell per round during any your Unit activation
            - Spell damage removes Paralysis token (friendly fire on)
            - IF vs Neutrals => +1 round costs 1 MP
        UNIT TURN
            1. Activate (mark Unit with your Faction Cube)
                a) May surrender for 10 gold IF NOT defending Town
                b) Remove Defense Token (can't use Defense Token this round)
                c) IF Paralysis token => remove Paralysis token & end Unit turn
                d) Use <Activate>, <Ongoing>, <Instant>
            2. Move
                - IF Ground/Flying => Move 0-3 then Attack/Defend
                - IF Ranged => Attack then Move 0-1
                               OR Defend only
                - IF Defense Token used => end turn
            3. Use <Activate>, <Ongoing>, <Instant>
            4. Attack Phase
                a) Attacker - Use <Instant>
                b) Attacker - Roll Attack die (D3: -1, 0, +1)
                            - IF Ranged in melee OR backline to backline => 2x Attack die, resolve lower
                c) Defender - IF Defense token => roll Defense die (D3: 0, 0, +1)
                d) Defender - IF Paralysis token => remove Paralysis token & Retaliate IF can
            5. Resolve damage
                - IF Gate/Wall attacked by Ground/Flying => Gate/Wall destroyed
                - DAMAGE =   Attack stat
                           - Defense stat
                           + Attack die
                           - Defense die
                           - 1 IF ranged attack & enemy Unit behind Wall/Gate
                - IF  DAMAGE < 0  =>  DAMAGE = 0
                - HP = HP - DAMAGE
            6. Retaliation
                - Only IF melee
                - Function like Attack Phase
                - Each Unit retaliates once per Combat Round (mark with Black Cube)
    COMBAT END
        - Discard all <Ongoing> in play
        RETREAT happens IF combat was not extended vs Neutrals except Azure
            - Move Hero to previous map field
            - Keep survived Units
        SURRENDER may happen on your Unit Activation only vs Hero IF NOT defending Town
            - Pay 10 gold to Enemy Hero
            - IF Main Hero surrenders => Move Hero to any your Town/Settlement IF any
            - IF Secondary Hero surrenders => Remove Hero
            - Keep survived Units
        DEFEAT happens IF all Units defeated OR Secondary Hero was removed before combat
            - IF Main Hero defeated => Gain Negative Morale & Move Hero to any your Town/Settlement
            - IF Main Hero defeated vs Hero => Pay 5 gold to Winner
            - IF Main Hero defeated without Town/Settlement => Lose game
            - IF Main Hero defeated when defending last Town/Settlement => Lose game
            - IF Secondary Hero defeated => Remove Hero
            - IF all Units defeated => Take starting Units
            - Winner Hero gains enemy Faction Cube & scenario bonus if any
            - Winner Hero gains Exp:
                - IF Hero level = Enemy level => +1 Exp
                - IF Hero level < Enemy level => +2 Exp
                - IF Azure enemy => gain 7 level
